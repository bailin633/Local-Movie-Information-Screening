常见问题解决方案

问题1: Python运行时不可用

  • 确认已安装Python 3.7或更高版本
  • 检查Python是否添加到系统PATH环境变量
  • 尝试重启应用程序
  • 使用上方的手动设置功能指定Python路径

问题2: 扫描目录失败

  • 确认Python可以正常运行
  • 检查目录权限
  • 尝试减少扫描深度
  • 查看详细错误信息

推荐的Python安装方式:

  • 从 python.org 下载官方安装包
  • 安装时勾选"Add Python to PATH"选项
  • 选择"Install for all users"（如果有管理员权限）
